Ubuntu相关事项：

重装系统：
U盘插入时：
选择usb开头的文件选择后进行安装
安装时选择  sda/adb 表达是否要格式化U盘分区，不格式化则会出现双启动，双启动很卡，所以格式化

系统安装完成之后：
sudo apt-get install update(开始install之前先update)
sudo apt-get install vim
sudo apt-get install
